# 02 Advanced CSS: Portfolio

---

## User Story

```
AS someone looking for a Web Developer position
I WANT a potential employer to view my deployed portfolio of work samples
SO THAT they can review samples of my work and asses whether I am a good candidate for an open position
```

---

## Acceptance Criteria

```
GIVEN I want a potential employer to sample my previous work and contact me
WHEN the user loads my portfolio
THEN the user is presented with my name, avatar, and links to sections about me, my work, and how to contact me
WHEN the user clicks one of the links in the navigation
THEN the UI scrolls to the corresponding section
WHEN the user clicks on the link to the section about my work
THEN the UI scrolls to a section with images of my applications
WHEN the user is presented with my first application
THEN that application's image is larger in size than the others
WHEN the user hovers over an image of my work
THEN the title of the work is displayed (3 second wait)
WHEN the user clicks on the images of the applications
THEN the user is taken to that deployed application via a new tab
WHEN the user resizes the page or view the site on various screens and devices
THEN the user is presented with a responsive layout with adaps to the users viewport
WHEN the user clicks on the link to the section about my contact
THEN UI scrolls to the corresponding section
WHEN the user hovers over one of the image icons on the contact section
THEN the title of the icon is displayed
WHEN the user clicks on either the GitHub or LinkedIn image icon
THEN the user is taken to the either my GitHub page or LinkedIn page via a new tab
WHEN the user clicks on the mail icon
THEN a new email window is opened on the users PC
WHEN the user clicks on the TOP button (bottom right corner)
THEN the user is taken to the top of the page
```
